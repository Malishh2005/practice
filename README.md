# practice

Інтерактивний фізичний симулятор зворотного маятника на візку, що використовує **нечітку логіку (Fuzzy Logic)** для стабілізації та методи глобальної оптимізації для пошуку найкращих параметрів керування.

##  Ключові особливості

* **Подвійний фізичний рушій:** Можливість порівняння точності обчислень між методом **Ейлера** та **Рунге-Кутта 4-го порядку (RK4)**.
* **Інтелектуальне керування:** Реалізовано нечіткий регулятор (Fuzzy Controller) з 9 базами правил, що враховує кут, кутову швидкість, позицію та швидкість візка.
* **AI Оптимізація:**
    * **Genetic Algorithm (GA):** Еволюційний пошук оптимальних коефіцієнтів через селекцію, схрещування та мутацію.
    * **Particle Swarm Optimization (PSO):** Моделювання поведінки зграї часток для швидкої збіжності до стійких параметрів.
* **Реалістичні умови:** Симуляція затримок мікроконтролера, шуму датчиків та зовнішніх збурень (вітер, поштовхи).
* **Аналітика:** Експорт історії успішних навчань у формат **CSV** для подальшого аналізу в Excel або Python.

##  Технологічний стек

* **Frontend:** HTML5 Canvas, Vanilla JavaScript (ES6+ Modules)
* **Math:** Математичне моделювання диференціальних рівнянь руху системи Лагранжа 2-го роду.
* **Data Vis:** Chart.js для візуалізації (опційно), динамічна зміна DOM для статистики.

##  Як працює Навчання

Під час натискання кнопки "Навчити", алгоритми (GA або PSO) запускають сотні "віртуальних тестів" у фоновому режимі. 
1.  Кожна частинка або індивід отримує свій набір коефіцієнтів $K_{\theta}, K_{\omega}, K_{out}, K_{x}, K_{v}$.
2.  Система оцінює **Fitness** на основі середньоквадратичного відхилення кута від вертикалі та зусиль, витрачених на стабілізацію.
3.  Найкращі параметри автоматично застосовуються до активного контролера.



##  Запуск проєкту

Проєкт працює як статична веб-сторінка. Для коректної роботи ES-модулів використовуйте будь-який локальний сервер (наприклад, Live Server у VS Code):

1. Склонуйте репозиторій.
2. Запустіть локальний сервер у кореневій папці.
3. Відкрийте `index.html` у браузері.
